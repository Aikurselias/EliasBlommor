Vanligt fel & fix-instruktioner

Om sidan hänger eller visar "Unknown layer: RandomFlip":

  1. Gå till GitHub → tfjs_model → model.json → Edit
  2. Ta bort hela Sequential-blocket med RandomFlip/RandomRotation/RandomZoom/RandomContrast
  3. Ändra inbound_nodes på mobilenetv2-lagret till [["input_2", 0, 0, {}]]
  4. Commit → vänta → Ctrl+F5 här

Om det fortfarande hänger: testa i annan webbläsare eller på dator istället för mobil.
