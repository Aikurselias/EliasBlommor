Viktigt – om modellen inte laddas (vanligt fel)

Du får troligen felet "Unknown layer: RandomFlip" eftersom augmenteringslagren (RandomFlip, RandomRotation, RandomZoom, RandomContrast) ligger kvar i model.json.

Snabbaste lösningen just nu:

  1. Öppna tfjs_model/model.json i GitHub (redigera filen direkt i webbläsaren)
  2. Leta efter blocket som börjar med "class_name": "Sequential" (augmenteringslagren)
  3. Ta bort hela det blocket (från {"class_name": "Sequential" ... till motsvarande slutklammer)
  4. Ändra "inbound_nodes" för nästa lager (MobileNetV2-delen) så det pekar direkt på input-lagret, t.ex.: 
    "inbound_nodes": [[["input_2", 0, 0, {}]]]
  5. Commit ändringen → vänta 1–2 min → uppdatera sidan (Ctrl+F5)

Alternativt: Träna om modellen i Python utan att ha augmenteringslagren i den slutgiltiga modellen som exporteras.
