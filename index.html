Om modellen

  • Bas: MobileNetV2 (lättviktig CNN)
  • Tränad med: Keras → konverterad till TensorFlow.js
  • Modellstorlek: ~8 MB (model.json + shards)
  • Klasser: daisy, dandelion, rose, sunflower, tulip
  • Körs 100% i webbläsaren – ingen server behövs

GitHub-repo: github.com/Aikurselias/EliasBlommor

Tips: Om du har stora modellfiler, ladda upp dem som ZIP-arkiv till GitHub och extrahera till tfjs_model-mappen.
